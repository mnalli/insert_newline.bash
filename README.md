# `insert_newline.bash`

Insert newline characters in `bash` command line as you would do in `zsh` or `fish`.
With a simple `Alt-Enter`.

## Installation

### Copy into your `.bashrc`

```sh
curl https://raw.githubusercontent.com/mnalli/insert_newline.bash/refs/heads/main/insert_newline.bash >> ~/.bashrc

source ~/.bashrc
```

### `git clone`

```sh
git clone https://github.com/mnalli/insert_newline.bash.git --depth=1 ~/.insert_newline.bash
```

Add the following to your `.bashrc`:

```sh
source ~/.insert_newline.bash/insert_newline.bash
```

*Note*: instead of `~/.insert_newline.bash/` you can use the path you prefer.
